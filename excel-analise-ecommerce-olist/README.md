Visão Geral

Neste projeto desenvolvi uma análise exploratória completa utilizando dados reais de um e-commerce brasileiro.

O objetivo foi integrar múltiplas bases de dados, estruturar métricas de negócio e identificar padrões relacionados a vendas, pagamentos e logística.

A análise foi conduzida utilizando exclusivamente Excel, simulando um cenário corporativo onde a ferramenta é amplamente utilizada para tomada de decisão.

🎯 Problema de Negócio

Como podemos:

Entender o comportamento dos pedidos?

Identificar padrões de cancelamento?

Avaliar o impacto do frete no valor final?

Analisar a relação entre meio de pagamento e performance do pedido?

🗂 Estrutura de Dados

Trabalhei com múltiplas tabelas relacionais, integradas via order_id, consolidando informações como:

Status do pedido

Valores de produtos

Frete

Tipo de pagamento

Parcelamento

Localização de vendedores

🛠 Desenvolvimento
🔹 Tratamento de Dados

Importação e padronização de arquivos CSV

Correção de separadores decimais

Tratamento de inconsistências com SEERRO

Ajustes de formatação e tipos de dados

🔹 Integração de Bases

Uso de PROCV para consolidação das informações

Simulação de lógica semelhante a JOIN em SQL

Construção de base analítica unificada

🔹 Criação de Métricas

Valor total do pedido (produto + frete)

Diferença entre valor pago e valor calculado

Ticket médio

Média por tipo de pagamento

Percentual de cancelamento por meio de pagamento

🔹 Análise Exploratória

Cruzamento de variáveis com Tabelas Dinâmicas

Avaliação de performance por status

Análise percentual por linha

Identificação de possíveis padrões de risco

🔹 Visualizações

Gráfico de dispersão (Price vs Payment Value)

Gráfico de barras por status

Mapas de calor com formatação condicional

📈 Principais Análises Realizadas

Distribuição de pedidos por status

Comparação entre valor do pedido e valor efetivamente pago

Avaliação do impacto do frete

Análise de cancelamento por meio de pagamento

Exploração de possíveis inconsistências entre valores

🧠 Competências Demonstradas

Manipulação de bases relacionais no Excel

Engenharia de métricas de negócio

Uso avançado de funções (SE, PROCV, SEERRO, MÉDIA, SOMA)

Construção de análises cruzadas com Tabela Dinâmica

Interpretação de dados com foco estratégico

🚀 Conclusão

Este projeto demonstra minha capacidade de trabalhar com múltiplas fontes de dados, estruturar análises e gerar insights relevantes utilizando Excel como ferramenta analítica.

A abordagem adotada buscou simular um ambiente corporativo real, priorizando organização, clareza e raciocínio orientado a negócio.
