# PORTIFOLIO_DADOS

A ideia aqui não é só mostrar código, mas demonstrar como eu penso, analiso problemas e transformo dados em informações úteis para tomada de decisão.

🚀 Sobre mim

Sou estudante de Ciência da Computação e estou em busca de uma oportunidade de estágio na área de Dados.

Tenho interesse em Análise de Dados, Business Intelligence e Ciência de Dados, sempre buscando aplicar técnica com visão de negócio.

Power BI – modelagem, DAX e visualizações estratégicas
