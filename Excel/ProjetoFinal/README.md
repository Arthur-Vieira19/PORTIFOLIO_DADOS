Projeto Origin: Análise de Vendas e Performance
O Projeto Origin é uma solução de inteligência de dados desenvolvida em Excel/CSV para gerenciar e analisar o ecossistema de vendas de uma operação comercial. O projeto foca na integração de múltiplas bases de dados (pedidos, itens, pagamentos e vendedores) para extrair insights estratégicos sobre faturamento e desempenho.

💻 Sobre o Projeto
Este projeto simula um ambiente real de análise de dados, onde informações dispersas são consolidadas para responder a perguntas de negócio. A estrutura permite monitorar o ciclo de vida de um pedido, desde a escolha dos itens até a confirmação do pagamento e a atribuição da comissão ao vendedor.

Funcionalidades Principais:
Gestão de Pedidos: Controle de status e datas de venda.

Análise de Itens: Detalhamento de produtos vendidos por transação.

Fluxo de Caixa: Monitoramento de métodos de pagamento e liquidez.

Performance de Vendas: Ranking de vendedores e produtividade por região.

Dashboards Dinâmicos: Utilização de Tabelas Dinâmicas para visualização rápida de métricas (KPIs).

🛠 Estrutura dos Dados
O projeto é composto por cinco bases principais interconectadas:

Pedidos (Orders): Cabeçalho das vendas com datas e identificadores.

Itens (Items): Detalhes dos produtos, quantidades e preços unitários.

Vendedores (Sellers): Cadastro da força de vendas.

Pagamentos (Payments): Métodos de pagamento (Cartão, Boleto, etc.) e parcelamento.

Dinâmicas: Relatórios consolidados para análise de tendências.

🚀 Tecnologias e Ferramentas
Excel / CSV: Manipulação e armazenamento dos dados.

Power Query (Opcional): Para limpeza e transformação dos dados.

Análise Estatística: Cálculo de médias, somatórios e variações percentuais.

📈 Insights Gerados
Através das planilhas de Dinâmicas, o projeto responde a:

Qual o ticket médio por categoria de produto?

Qual vendedor possui a maior taxa de conversão?

Quais são os períodos de pico de vendas (sazonalidade)?

Qual a forma de pagamento preferida dos clientes?
