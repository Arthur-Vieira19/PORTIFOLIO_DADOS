# 📈 Projeto Origin — Análise de Vendas e Performance

Solução de análise de dados em Excel para consolidar
múltiplas bases de uma operação comercial e extrair
insights estratégicos sobre faturamento, comportamento
de pagamento e performance de vendedores.

---

## 🎯 Problema

Operações comerciais com dados dispersos em múltiplas
planilhas perdem visibilidade sobre o ciclo de vendas.
Este projeto integra 4 bases relacionadas para responder
perguntas de negócio de forma centralizada.

---

## 🗂️ Modelagem dos Dados

| Tabela | Descrição |
|---|---|
| Orders | Cabeçalho dos pedidos — datas e status |
| Items | Produtos, quantidades e preço unitário |
| Sellers | Cadastro e região da força de vendas |
| Payments | Método de pagamento e parcelamento |

Relacionamento entre tabelas via chave de pedido (Order ID),
simulando modelagem relacional em ambiente Excel.

---

## 🔍 Insights Encontrados

- Ticket médio por categoria: **R$ 115,01** *(preenche)*
- Pico de vendas concentrado em **4º TRIMESTRE**
- Método de pagamento dominante: **CARTAO DE CREDITO** representa 75,44%

---

## 🛠️ Técnicas Aplicadas

- **Power Query** — limpeza e transformação das bases
- **Tabelas Dinâmicas** — KPIs consolidados por dimensão
- **Funções Excel** — PROCV, SOMASE,  MÉDIASE, TABELA DINAMICA
- **Análise estatística** — ticket médio, variação percentual,
  ranking de performance
